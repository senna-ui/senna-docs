# Installation

Senna UI React is available as an [npm package](https://www.npmjs.com/package/@senna-ui/react)
To install and save in your `package.json` dependencies, run:

```sh
// with npm
npm install @senna-ui/react

// with yarn
yarn add @senna-ui/react
```
